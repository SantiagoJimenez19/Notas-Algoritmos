# Notas-Algoritmos
Aquí estarán mi notas de la clase de Algoritmos en el curso **Enero-Junio 2026**
Y explicaré de forma superficial en que consiste cada uno
Muchos son más faciles de visualizar en una gráfica $G$

## Agente Viajero
El problema consiste en pasar por todos los puntos de un pueblo, con la menor cantidad de puntos

## Número cromático
Buscar el número minimo de colores o etiquetas tales que ningún color es adyacente entre sí $\chi(G)$. El problema se presento en termino de horarios

## Emparejamiento
Determinar si se puede encontrar un emparejamiento, respetando las elecciones de cada vertice, pero en realidad el problema se reduce a determinar si la gráfica correspondiente es bipartida.

### Prioridad
Es una variante en la que cada persona propone una lista de prioridad

## Sublista
Dada una lista, se busca la sublista con mayor suma (Una lista es todo un *intervalo*, no un subconjunto)

## Árbol Generador
Se presento como un problema en donde se busca conectar una región o conjunto de pueblos, con el menor costo posible (por lo que los ciclos no están permitidos, ya que representarían una perdida)

## Gráficas Hamiltonianas
Determinar si una gráfica es *Hamiltoniana*, es decir, si se puede recorrer todos los vertices una sola vez, culminando en el que comencé.

## Teorema de los 4 colores
Todo mapa puede ser coloreado con 4 colores, sin que dos paises adyacentes tengan el mismo color. (se traduce a que toda gráfica plana tiene $\chi(G)=4$)

Estos solo son algunos de los problemas clasiscos.
